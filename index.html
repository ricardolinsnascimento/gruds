Projetos Mitã

O projeto Mitã é resultado de uma parceria entre IFMS e APAE. Desenvolvido como produto de TCC das estudantes Liliane Dauzacker Gomes e Daynara Martins da Silva, sob orientação do Professor Ricardo Nascimento, o projeto é um conjunto de jogos que ajudam no ensino de matemática para crianças com deficiência. Desenvolvido com HTML, CSS e JavaScript
